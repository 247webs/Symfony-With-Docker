# Blog Post API

### Description
In this project we are creating article with authorized user. On  the user article other user can comment or also follow the article user

### Requirements

| Package | Version |
|---|---|
| PHP  | 7.4.3 |
| MySQL | mysql |
| Apache | Apache/2.4.46 (Ubuntu) |
| Symphony | 5.4 |

## Local setup
```bash
$ git clone {REPOSITORY_URL}
$ cd blog-post-api
```

On first run :

```bash
$ make install
```

On next runs :

```bash
$ make start
```

# Run phpunit tests

```bash
$ make test
$ make specs
```
